SIMULATE AND PLOT UNRESTRICTED TRAJECTORIES
for
BW AND HO
potentials.

Execute e.g.
python LE_PlotII.py 0.1 1.0 --BW --sm 1.0 -t 1.0

Will search for / generate TRA file using LE_Simulate.py.
Then produce PDF and STR plots using LE_PlotII.py.

LE_PlotII makes streamplot from PDF data.
LE_Plot makes streamplot from TRA data.

Note TRA files can be large, and as such have been moved out of
Dropbox.
